Spring Boot部署

　　Spring Boot 应用支持jar包和war包两种方式

1). jar包方式（maven）

　　编写好Spring Boot应用后，在项目目录中打开cmd命令行，输入mvn install ，命令运行完毕后，在target目录下会生成一个名为springboot-0.0.1-SNAPSHOT.jar的jar包。

　　命令行进入/target目录，使用java -jar springboot-0.0.1-SNAPSHOT.jar便可使其运行，浏览器中输入:127.0.0.1:8080/xxx 即可看到服务器返回的结果。

3）Spring Boot热部署

　　当发生任何修改时，需要重启Application类才能生效，略显麻烦。Spring Boot支持热部署，当发现任何类发生了改变，马上通过JVM类加载的方式，加载最新的类到虚拟机中。 这样就不需要重新启动也能看到修改后的效果了。
